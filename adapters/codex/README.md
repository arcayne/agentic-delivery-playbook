# Codex adapter

Use this adapter as Codex repository instructions.

## Install in a repository

Copy `AGENTS.md` into the repository root:

```bash
cp adapters/codex/AGENTS.md /path/to/your/repo/AGENTS.md
```

If the repository already has an `AGENTS.md`, merge the playbook rules into the existing file instead of overwriting project-specific instructions.

## One-off use

If your Codex setup does not read `AGENTS.md`, paste the contents of `AGENTS.md` at the start of the session and then describe the task.
